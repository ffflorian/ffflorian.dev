# ffflorian.dev

My personal website, built with [Hugo](https://gohugo.io/) and my [modified version of the PaperMod theme](https://github.com/ffflorian/hugo-PaperMod).

## Requirements

* [go](https://go.dev/)
* hugo (`go install github.com/gohugoio/hugo@latest`)

## Start locally

```
hugo server
```
